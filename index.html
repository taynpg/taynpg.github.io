Go环境

GOPATH

在指定GOPATH 工作目录，代码总是会保存在 $GOPATH/src 下。在工程经过 go build、go install 或 go get 等指令后，会将产生的二进制可执行文件放在 $GOPATH/bin 目录下，生成的中间缓存文件会被保存在 $GOPATH/pkg 下。
